
This repository serves as a workspace for experimenting with various AI models and libraries, and it will be continuously updated with new projects and features in the future.

## 📂 Current Projects

The repository is currently organized into several distinct computer vision modules:

* **`face-analysis`**: Scripts to analyze facial features and attributes from static images.
* **`face-recognition`**: Identifying and matching faces within static images.
* **`face-recognition(video)`**: Real-time face recognition applied to video streams.
* **`gestures`**: Implementation of hand landmark detection to map out hand structures.
* **`gestures-recognition`**: Identifying and classifying specific hand gestures using trained AI models.

## 🛠️ Tech Stack & Tools

* **Language**: Python 
* **Models/Assets**: Includes custom `.task` models (like `hand_landmarker.task` and `gesture_recognizer.task`) for on-device machine learning tasks and etc....
